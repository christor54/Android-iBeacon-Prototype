This project present a prototype based on Android that interacts with iBeacons (with a specific UUID) and display specific content considering in which beacon range you are.

It contains two parts :

-beacdroid which is the Android app that recognize the beacons around and display specific web content when entering into a zone range of one. 
The content to be displayed and the ranges are defined in the json file beacon.json

-beacdroid.com which contains the web content to be display (web pages with image and audio)

Technologies used in this project : Android, altbeacon library,JSON, HTML5, CSS3
